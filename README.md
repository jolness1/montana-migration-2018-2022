# Montana American Community Survey migration

## Purpose:
- Quantify who moved to Montana over the 2018-2022 period

## Inputs
- Used [2018-2022 American Community Survey](raw-inputs/state-to-county-migration-flows-acs-2018-2022.xlsx) 
    - to create [Montana migration data](data/2018-2022-american-community-survey-MT.csv) (used for analysis)
    - source: [U.S. Census State to State Migration](https://www.census.gov/data/tables/time-series/demo/geographic-mobility/state-to-county-migration.html)
- A list of [U.S. states & territories](data/states.txt) to separate US migration from Foreign migration
